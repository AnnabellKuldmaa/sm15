This is a simple command line application for conformance checking.

#For running from command line:
$ ant run

Application asks for full path of the Petri net file (.pnml)
and the log file (.xes).

The fitness, behavioral appropriateness and structural appropriateness will be calculated up to 
two decimal places and displayed.
